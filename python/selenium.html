Selenium Webdriver 简易教程 - wizardforcel

某些网站的反爬虫机制通过按钮、跳转等功能，能够防止最简单的 requests 爬虫。

Selenium WebDriver是一个测试框架，提供了 Java、Python 等语言的 API，可以用一种更底层、更灵活的方式来操作浏览器，模拟人在浏览器上的操作。可用于 Web 应用的测试，当然也可以用于自动化批量操作（管理员批量部署、爬虫）等等。

同时，需要下载 ChromeDriver 或指定 FireFox 浏览器的路径。
